# Acknowledgments

This project exists because of the open-source work of others. Thanks to:

- **[JMRI](https://www.jmri.org/)** — the model railroad control platform this
  entire project talks to via its JSON Web Service. Without JMRI's Web Server
  and JSON API, this project wouldn't have a server to control.
- **[DCC++](https://github.com/DccPlusPlus)** — the open-source DCC command
  station firmware running the layout this project was built and tested
  against.
- **[xiaozhi-esp32](https://github.com/78/xiaozhi-esp32)** — the open-source
  voice assistant project whose MCP pipe example `xiaozhi_wrapper` is adapted
  from. See the [README](README.md#third-party-code) for the full MIT license
  notice.

See [docs/resources.md](docs/resources.md) for links to these and other
projects referenced throughout this codebase.
